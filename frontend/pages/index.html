Server-Side Timers

60-second countdowns run on the server via Socket.io timer-tick events. Zero client manipulation possible.

🎲

MongoDB $sample Shuffle

Each session uses aggregate([{ $sample: { size: 15 } }]) to pick unique random questions at the database level.

🏆

Daily Leaderboard

Scores filtered from midnight today. Rankings update in real-time via Socket.io new-score-update broadcasts.

🔐

JWT + Google OAuth

All quiz routes secured with JWT. Google Sign-In via Passport.js. Rate limiting on all sensitive endpoints.

🎯

Time Bonus System

Faster correct answers earn more points. Up to +5 bonus per question based on remaining server time.

🏗️

MVC Architecture

Clean separation: Controllers, Models, Routes, Middleware, Utils, Socket layer. Production-grade Node.js structure.
